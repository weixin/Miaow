# WeSketch [![Version Number](https://img.shields.io/github/release/weixin/WeSketch.svg?style=flat)](https://github.com/weixin/WeSketch/ "Version Number")

> 一个强大 Sketch 插件合集，由微信设计团队量身打造，让你的视觉设计、交互设计、网页重构更加高效快捷。

## 下载&安装

[下载地址](https://github.com/weixin/WeSketch/releases)   
(支持 Sketch 44+)
下载解压后，双击 `wesketch.sketchplugin` 安装
