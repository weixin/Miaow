# WeSketch 

[下载解压后双击wechat.sketch文件安装](https://github.com/weixin/weBox/archive/master.zip)

wechat sketch插件，提升设计工作效率
sketch 已经成为移动端 ui 设计师标配，weBox 用于解决 sketch 最后一点不好用的地方。
